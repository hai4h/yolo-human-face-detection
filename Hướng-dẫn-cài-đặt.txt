###  Yêu cầu hệ thống

- Hệ điều hành Ubuntu(hoặc tương tự có hỗ trợ các dependencies)
- Python 3.10+
- Thư viện Python: FastAPI, Uvicorn, PyMongo, TensorFlow, OpenCV, Ultralytics (YOLO)
- Trình duyệt web hiện đại (Chrome, Firefox, Edge, Safari) có hỗ trợ WebRTC
- MongoDB cài đặt trên máy tính cục bộ(sử dụng cách khác sẽ phải cấu hình lại trong mã nguồn)

### Cài đặt

1. Clone mã nguồn từ kho lưu trữ:
   ```
   git clone https://github.com/hai4h/yolo-human-face-detection.git
   cd yolo-human-face-detection
   ```
2. Cài đặt Framework hỗ trợ sử dụng GPU để tăng tốc xử lý(hãy chắc chắn rằng phần cứng được hỗ trợ)
    ```
    python3 -m venv .venv/
    source .venv/bin/activate
    ```

    - AMD GPU (ROCM):
    ```
    pip install tensorflow-rocm==[wheel-version] torch==[wheel-version] torchvision==[wheel-version] -f [repo] --upgrade
    ```
    [wheel-version] là phiên bản cần cài đặt tương ứng trong repo.
        VD: torch-2.4.0+rocm6.3.0-cp312-cp312-linux_x86_64.whl sẽ tương ứng torch==[2.4.0+rocm6.3.0]

    [repo] là https://repo.radeon.com/rocm/manylinux/rocm-rel-X.Y/, với X.Y là phiên bản của ROCm hiện có trên máy


    - NVIDIA GPU (CUDA):
    ```
    pip install tensorflow-cuda
    pip3 install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu118
    ```


3. Cài đặt các thư viện Python:
   ```
   pip install -r ultralytics fastapi uvicorn opencv-python
   ```

4. Chạy file run_server.py để khởi động hệ thống, port của hệ thống được định nghĩa trong run_server.py
